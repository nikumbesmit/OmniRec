# OmniRec

## Problem Statement

Current recommendation systems usually recommend only one type of content such as movies, books or music.

The goal of OmniRec is to build a unified recommendation platform capable of recommending movies, books and music using hybrid recommendation techniques.

The system should provide:

- Personalized recommendations
- Similar item recommendations
- Cross-domain recommendations
- Explainable recommendations
- Cold-start support
- Scalable API-based architecture

The project is designed using Machine Learning Engineering best practices including modular pipelines, model evaluation, API deployment and production-ready architecture.
